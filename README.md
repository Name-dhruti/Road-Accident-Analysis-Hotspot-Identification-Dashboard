Road Accident Analysis & Hotspot Identification Dashboard

Project Overview :-
This project analyzs road accident data to identify accident treds, high-risk areas(hotspot), and major contributing factors affecting road safety. Using Power BI , the dashboard transform raw accident records into meaningful insights thrugh interactive visualizations and KPIs.
The dashboard helps users understand:
- When accidents occur most frequently
- Which areas are accident-prone
- Key risk factors contributing to accidents
- Accident severity distribution
- High-risk zones requring safety improvements

Tools Used :-
- Power Bi
- Power Query
- DAX

Dataset Information :-
It consists of 12,316 accident records and 32 attributes.
Dataset Features:
- Driver Information: Age,gender,education level,driver experience and driver-vehicle relationship.
- Vehicle Information:Vehicle type,owenership,server year,defects,and number of vehicles involve.
- Road Information:Accident area,road alignment,junction type,lane/median details,and road surface conditions.
- Environment Factors:Weather coditions,light conditions,day of thee week,and accident time.
- Casualty Information:Casualty class,age group,gender,severity,accupation,and fitness status.
- Accident Details:Type of collision,vehicle movement,cause of accident,and accident severity.

Dataset Cleaning Process :-
- Handled missing and null values
- Standardized categorical data
- Created calculated columns and DAX measures
- Extracted hour information from time data
- Validated and formatted data for analysis

Key Insights :-
- Most accidents occur during peak evening hours due to high traffic density.
- Urban areas such as office and residential zones experience the highest accident frequency.
- Careless driving and failure to maitain safe distance are major accident causes.
- Most accidents are classified as slight severity,though serious and fatal incidents are still significant.
- Weather and road surface conditions infuence accident occurence but driver behavior remains the dominant factor.
